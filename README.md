# 6nimmt
### Usage
```
usage: simulate.py [-h] [-n N] [-p P] {2p_rel,g_pos}

positional arguments:
  {2p_rel,g_pos}

options:
  -h, --help      show this help message and exit
  -n N            Rounds played
  -p P            Number of players
```


### Ranking
`python simulate.py g_pos`

### Matrix
`python simulate.py 2p_rel`
